# COMERCIAL

Esse relatório é o resumo de receita e lucro e demais informações.

📊 Power BI – Relatório de Comercial. 
____________________________________________________________________________________________________________________________________

Este projeto apresenta um dashboard desenvolvido no Power BI para uma empresa fictícia, analisando a performance de receita e lucros da empresa.
_____________________________________________________________________________________________________________________________________

🔎 Visão Geral do Projeto
_____________________________________________________________________________________________________________________________________
O objetivo do relatório é oferecer uma análise clara e visual sobre as receitas e lucros da empresa do modo geral e detalhado.
_____________________________________________________________________________________________________________________________________

📈 Principais Análises
_____________________________________________________________________________________________________________________________________
* Números de Receita total (Por: continente gerais, continente detalhado, subcategoria)

* Números de Clientes( Compras, lucros, idade).

* 🛠️ Ferramentas Utilizadas
_____________________________________________________________________________________________________________________________________
* Power BI Desktop
* Power Query
* Linguagem DAX
* Modelagem e tratamento de dados
